1. 회원가입
  아이디 (중복 확인)
  비밀번호
  비밀번호 확인 (비밀번호 체크)
  이름
  직책
  전화번호 

  위 사항 체크 후 회원 가입 완료
  완료 되면 로그인하러가기 버튼이 뜨고 클릭 시 로그인화면으로 이동

2. 로그인
  아이디, 비밀번호 입력
  안맞을 경우 이상 문구 발생

  로그인 성공 시 메뉴바 우측에 이름(직책) 표시 되고
  로그아웃 활성화
  로그아웃 클릭 시 로그아웃 됨.

3. PIMS 요청사항에 들어감
  운영자인 경우 글 작성 가능 (개발자는 안됨)
  개발자인 경우 글 조회에서 상태 변경 가능 (처리,완료,지연,불가)
  상태에 따라 표에서 색상 표시 됨.

  운영자, 글 작성 시 폰트, 굵기 변경 가능
  표 삽입, 이미지 삽입 가능
